    expire_cache() click to toggle source

    Includes header tags that instruct browsers not to cache the page. Ajax methods that update pages (eg: change_chronicle) update the page but hitting the back button in a browser will display un-updated contents.

    Calling this method prevents that behavior.

    # File app/controllers/application_controller.rb, line 35
def expire_cache
  response.headers["Cache-Control"] = "no-cache, no-store, max-age=0, must-revalidate"
  response.headers["Pragma"] = "no-cache"
  response.headers["Expires"] = Time.now.httpdate
end

def obsidian_enabled?
  not SERVICES['obsidianportal']['consumer_key'].empty? and user_signed_in? and not current_user.obsidian_user_id.nil?
end
